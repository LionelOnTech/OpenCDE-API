title Upload New Document

note over Client,Browser: Start upload document flow
Client->Browser: BROWSER /cde/0.1/documents/upload/?callback_url=http://localhost/ClientCallback&filename=<filename>
note over Browser,CDE: User enters required upload document fields
Browser->CDE: Request upload document page
CDE->Browser: CDE upload document revision page
Browser->CDE: Populated upload document version fields
CDE-->Browser: redirect (with CDE specific payload of upload fields)
Browser-->Client: CDE payload (not inspected by client)
note over Client,CDE: Upload the document
Client->CDE: POST /documents-api/documents (file binary and CDE payload)
CDE->CDE: Persist document in CDE
CDE->Client: Document reference
